# 🎄 크리스마스 편지

특별한 사람에게 따뜻한 크리스마스 편지를 보내는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 📝 크리스마스 편지 작성
- 🔗 공유 가능한 URL 생성
- ❄️ 눈 내리는 애니메이션 효과
- 🎨 아름다운 크리스마스 테마 UI
- 📱 모바일 반응형 디자인

## 🚀 로컬에서 실행하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어보세요.

## 📦 Vercel로 배포하기

### 방법 1: Vercel 웹사이트 사용 (추천)

1. [Vercel](https://vercel.com)에 가입/로그인
2. "New Project" 클릭
3. GitHub 저장소 연동
4. 이 프로젝트 선택
5. "Deploy" 클릭
6. 배포 완료! 🎉

### 방법 2: GitHub 연동 자동 배포

1. GitHub에 저장소 생성
2. 코드 푸시:
```bash
git init
git add .
git commit -m "Initial commit: Christmas Letter App"
git remote add origin [your-github-repo-url]
git push -u origin main
```
3. Vercel에서 GitHub 저장소 import
4. 자동으로 배포됨!

### 방법 3: Vercel CLI 사용

```bash
# Vercel CLI 설치
npm i -g vercel

# 배포
vercel

# 프로덕션 배포
vercel --prod
```

## 🎁 사용 방법

1. 메인 페이지에서 편지 작성
2. "링크 생성하기" 버튼 클릭
3. 생성된 URL 복사
4. 친구에게 URL 공유
5. 친구가 링크를 열면 크리스마스 편지 확인!

## 🛠️ 기술 스택

- **Next.js 15** - React 프레임워크
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 스타일링
- **Vercel** - 배포 플랫폼
